Hardware Information

Hyperconverged compute and storage
  • 16 primary compute nodes
  • Two AMD EPYC 7402 CPUs at 2.8 GHz each
  • 48 cores and 96 threads per node (24 cores per CPU)
  • 512GB DDR4 SDRAM with a transfer rate of 3200 Mt/s on each node
  • Two 960GB SSD for disk storage on each node
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 10 Gbps connection
Home file system
  • /home NFS is made from the NFS machine
  • 192GB DDR4 SDRAM with a transfer rate of 3200 Mt/s
  • Twenty-four 3.2 TB NVMe SSDs
  • Two 240GB m.2 BOSS RAID cards in RAID-1
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 10 Gbps connection
Storage system
  • /data CephFS is made from disks on each compute node
  • 1.1 PB usable storage capacity
  • Twelve 16TB disks on each compute node for Ceph OSDs
  • One 1TB m.2 SSD on each compute node for metadata
  • One 1TB m.2 SSD on each compute node used for RocksDB database
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 10 Gbps connection
Interactive
  • 6 interactive nodes
  • Two AMD EPYC 7402 CPUs at 2.8 GHz each
  • 48 cores and 96 threads per node (24 cores per CPU)
  • 256GB DDR4 SDRAM with a transfer rate of 3200 Mt/s on each node
  • Two 960GB SSD for disk storage on each node
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 10 Gbps connection
GPU server
  • Four NVIDIA HGX A100 GPUs
  • Two AMD 7543 CPUs at 2.8 Ghz each
  • 64 cores and 128 threads (32 cores per CPU)
  • 192GB DDR4 SDRAM with a transfer rate of 3200 Mt/s
  • Two 960GB SSDs for disk storage
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 10 Gbps connection
Network
  • Two Juniper QFX5120 analysis facility switches
    • 40 Gbps bandwidth each for a total bandwidth of 80 Gbps
    • 48 10/25GbE SFP28 ports each for 96 total ports
  • Two Juniper QFX10002 distribution switches
    • 100 Gbps bandwidth each for a total of 100 Gbps
    • MC LAG configuration creates a virtual single switch
    • Interfaces with Science DMZ
Fast Disk Compute
  • 8 fast disk compute machines
  • Two Intel Xeon Gold 6348 CPUs at 2.6 GHz each
  • 28C/56T for each processor, x2 per server
  • 384GB DDR4 SDRAM with a transfer rate of 3200Mt/s on each node
  • NVMe drives used for 6TB of fast /scratch disk each node
  • BOSS-S2 controller with two 240GB m.2 in RAID1 for root disk
  • Broadcom BCM57414 Dual port 10/25GbE SFP28
  • OCP NIC 3.0 25 Gbps connection
